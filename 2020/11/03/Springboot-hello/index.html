Springboot的第一个项目（helloworld）

学习开发一个新的东西，首先得先从hellowold开始！

第一步-创建

菜单 -> New -> Project -> Spring Initializr 然后点 Next

第二步-选择模块

选择web模块：接着左边选择 Web, 右边只勾选 Web 即可，然后点击Next

第三步：SpringbootApplication.java

项目创建好之后，就自带一个SpringbootApplication, 其被@SpringBootApplication 所标记，表示这个是一个Springboot 应用
